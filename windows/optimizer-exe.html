Win｜史上最强系统优化神器，大小仅2m

本文最后更新于 2024年4月13日 上午

和Mac电脑相比，Windows电脑用久了，系统难免会变得臃肿卡顿。为了解决这个问题，小伙伴们会选择系统优化工具进行清理和优化。

市面上有很多系统优化工具，但很多都是收费的，而且功能臃肿，或者加了很多私料。

今天涯木君和大家分享一款功能强大的 Windows 系统优化开源项目 - Optimizer。

介绍

Optimizer是一款面向 Windows 系统的优化工具，采用C#编写，旨在提升计算机性能、加强隐私和安全保护。

它支持22种语言，包括简体中文和繁体中文，同时提供了许多实用的功能，包括：

  • 关闭不必要的 Windows 服务
  • 停止 Windows 自动更新
  • 卸载 UWP 应用
  • 清理系统垃圾文件和浏览器配置文件
  • 修复常见的注册表问题
  • 提供硬件检测工具
  • 测试 IP 连通性和延迟
  • 快速更改 DNS 服务器
  • 编辑 HOSTS 文件
  • 识别和终止文件锁定句柄 …

因体积小巧，功能强大，使用方便，受到了很多用户的喜爱。

这款工具号称是win平台最好的优化工具，目前在Github上面有11.9k star，很受欢迎！

功能特色

  • 免费开源：完全免费，无任何广告和功能限制。
  • 体积小巧：仅有 2MB 左右，无需安装，解压即用。
  • 功能强大：提供多种实用功能，满足不同用户的需求。
  • 使用方便：操作简单，即使是小白用户也能轻松上手。
  • 安全可靠：代码经过严格测试，不会对系统造成任何损害。

安装使用

直接到项目release页面，下载exe文件即可，文件不大只有2m左右。

打开后，首页显示如下。功能很多，都是win系统的一些常用优化功能。

下面我再简单介绍几个功能。

  1. 关闭不必要的 Windows 服务

Windows 系统默认会启动很多服务，其中一些服务并不常用，但会占用系统资源。比如自动更新、小娜cortana等。这些都可以自定义关闭，从而提升系统性能。

  1. 清理系统垃圾文件

win系统在运行过程中会产生大量的垃圾文件，这些文件会占用磁盘空间。Optimizer 可以帮助清理这些垃圾文件，释放磁盘空间，提高系统运行速度。

  1. 修复注册表问题

注册表是win系统的一大特色，在使用过程中会出现问题，导致系统运行异常。Optimizer 可以帮忙一键修复。

另外，Optimizer 还提供了一些高级功能，如：卸载 UWP 应用、更改 DNS 服务器、编辑 HOSTS 文件等等，非常方便，感兴趣的小伙伴可以试试。

在使用 Optimizer 进行系统优化时，注意：

  • 备份重要数据，以免造成意外损失。
  • 谨慎使用高级功能，可能会对系统造成影响。
